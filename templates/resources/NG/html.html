Component Templates

Templates for components can be applied in one of two ways.

Inline Template


@Component({
  selector: 'app-comp',
  template: `
    
Hello Angular

  `
})
export class AppComponent {
/* . . . */
}

External Syntax


@Component({
  selector: 'app-comp',
  templateUrl: './app-component.html'
 })
export class HeroAppComponent {
/* . . . */
}
