Core HTML Tags

Doctype

The Doctype tag tells the web browser what version of HTML is contained in your page. 

                    
<!doctype html>

HTML Tag

Your HTML tag immediately follows the Doctype tag. Every page must have one and all content goes in between the tags. 

                    
<!doctype html>
<html>
</html>

Head and Body Tags

The Head tag contains information used by the browser or search information

The Body tag contains everything visible to the user 


<!doctype html>
<html>
    <head>
        <title>Page Title</title>
    </head
    <body>
        Page Content
    </body>
</html>
