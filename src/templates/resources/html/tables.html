Tables

Tables are use to display tabular data in a grid formation. Tables are frequently misused.

  • Base Tag:<table/>
  • Row Tag:<tr>
  • Column Tag:<td>

What the HTML looks like

                        <table>
                            <tr>
                                <td>Column 1</td>
                                <td>Column 2</td>
                                <td>Column 3</td>
                            </tr>
                            <tr>
                                <td>Row 2</td>
                                <td>Column 2</td>
                                <td>Column 3</td>
                            </tr>
                            <tr>
                                <td>Row 3</td>
                                <td>Column 2</td>
                                <td>Column 3</td>
                            </tr>
                        </table>
