Things to note

Everything between a {{ and }} signify a Handlebars expression. Handlebars has a set of keywords to use inside expressions. Any non-keywords get mapped from a JavaScript objec.t

Example


    var template = '
{{title}}

    var input = { title: 'testing'};

log helper

You can use a logging technique in your templates if you start running into issues.


  {{#each blog}}
    
  • {{title}}
    • 
    {{log "blog" blog}}
    {{log "Title" title}}
  {{/each}}

    This logs the blog object and title to the console so that you can see their values.
