Undefined

Undefined is the value that is given to a variable before it has a value. 


    var x;
    console.log(x); //undefined     
    x = 10;
    console.log(x); //10

Variable hoisting.


    console.log(x); //undefined     
    var x = 10;
    console.log(x); //10

Is actually


    var x;
    console.log(x); //undefined     
    x = 10;
    console.log(x); //10
