Command Line

The Super Basics

Per Platform

Mac: Terminal

Win: GitBash

Pro Tip

Use tab to auto complete commands and paths. It will save you a lot of time.

Listing a directory

ls


$ ls

MyForks/   presentations/  'year 1 slides'/

Changing a Directory

cd


#list directory
$ ls

MyForks/   presentations/  'year 1 slides'/
#change directory into MyForks
$ cd MyForks/
$ ls

project1/ project2/

Directory Shortcuts

  • . - Current Directory
  • .. - Parent Directory
  • ~ - User Home - C:\users\daniel.worthy
  • / - Root

Displaying your current path.

Got lost in your console? Use the print working directory command to get a full path of your current location

pwd


$ cd ~
$ pwd
/home/daniel.worthy

Creating a directory/folder

You can create a new folder by using the make directory command.

mkdir


#list directory
$ ls

MyForks/   presentations/  'year 1 slides'/

$ mkdir my-work 
$ ls

my-work/  MyForks/   presentations/  'year 1 slides'/

Make a file via command line

touch


#list directory
$ ls

my-work/ MyForks/   presentations/  'year 1 slides'/

$ cd my-work 
$ ls


$ touch index.html
$ ls

index.html

The Clear Command

You can clear the console by simply typing clear and hitting enter.
