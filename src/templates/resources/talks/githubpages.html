Publishing your Angular CLI app to GitHub pages can be a little challenging.

We can use a node package called Angular CLI GHPages

Let's install it!


npm i -g angular-cli-ghpages

Step 1 - The build command


  ng build --prod --base-href "https://worthyd.github.io/taco-truck/"
  • --prod - Compiles the project without the debugging code.
  • --base-href - sets the root of your website

Step 2 - The Publish


    angular-cli-ghpages
