Navigation Menus

All sites need them and they can get really complicated.

General Notes for Navs

  • Needs to include links to all major parts of your site.
  • Include a logo.
  • Always make sure the user knows to to return to the home page.
  • Don't overload menu with too many items.
  • Put less important items in the footer menu
    • Social Links
    • Privacy Policy
    • etc.
  • Use drop downs to group together similar pages.
  • Have a plan to handle everything on mobile.

Drop Down Menus

  • Can be done with pure CSS
  • Traditionally use hover events
  • Make sure they work properly on a touch device.
  • The more tiers you go, the harder it is for users.
