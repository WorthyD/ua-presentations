The Attribute Selector

The attribute selector allows you to select elements by their attributes or attribute values.

Example

[attr] { /*css*/ }

[target] { /*css*/ }

a[target] { /*css*/ }

Attribute Value Selector

[attr="value"] { /*css*/ }

[type="text"] { /*css*/ }

input[type="text"] { /*css*/ }
