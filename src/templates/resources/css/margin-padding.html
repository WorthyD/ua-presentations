Margins

CSS Margins add spacing around an element.

  • margin-left
  • margin-top
  • margin-right
  • margin-bottom

Margin Shorthand


.fig10 {
    /* Applies 5px to all 4 sides */
    margin: 5px;
}
.fig11 {
    /* Applies 5px to all 4 sides */
    margin: 5px 5px 5px 5px;
}
.fig12{
    /*Applies 10px to top and bottom
      and 30px left and right*/
    margin:10px 30px;
}

Padding

CSS Padding add spacing inside an element.

  • padding-left
  • padding-top
  • padding-right
  • padding-bottom

Padding Shorthand


.fig13 {
    margin: 5px;
    padding:15px;
}
.fig14 {
   margin:10px 30px;
   padding:30px 10px;
}
