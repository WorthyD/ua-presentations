Height & Width

Sets the specific height and width of a block element

  • height
    • min-height
    • max-height
  • width
    • min-width
    • max-width

Examples


.fig15{
    height:100px;
    width:250px;
}
/*Applying this to a span tag*/
.fig16{
    height:100px;
    width:100px;
}
.fig17{
    max-width:100px;
}
