User Interaction

We wire up our interaction events in our template.

Basic Events

  • clock
  • keyup
  • blur

Click Event Wireup


@Component({
  selector: 'click-element',
  template: `
    
    {{clickMessage}}`
})
export class ClickElementComponent {
  message = '';

  clickFunction() {
    this.message = 'I was clicked!';
  }
}
