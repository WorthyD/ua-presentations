ngFor

ngFor allows you to loop through and display an array of objects in the view.

Syntax


export class AppComponent {
  names = ['Ted', 
    'Fred', 
    'Marco', 
    'Polo'];
}


    
    
  • 
        {{name}}
    
    •

Output



    
    
  •  
        Ted
    
    • 
    
  •  
        Fred
    
    • 
    
  •  
       Marco 
    
    • 
    
  •  
        Polo
