Interfaces In the CLI

Interfaces provide the easiest way to provide consistency across your application.

Interface Generation


ng generate interface my-interface-name

#or

ng g i my-interface-name

Only generates .ts file.

Interface Structure


export interface MyInterfaceName {
    id:string;
    name: string;
    description:string;
}

Interface Usage


import { Component } from '@angular/core';

import { MyInterfaceName } from './my-interface-name';

@Component({
  selector: 'my-app',
  template: `
`
})
export class AppComponent { 
    MyItem: MyInterfaceName;
}
