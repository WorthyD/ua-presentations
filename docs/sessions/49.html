ngIf

There are times where you may need to hide or show content based off a condition in JavaScript. In the past, we've use CSS and classes to hide elements. We can use ngIf to show or hide content.

NgIf actually adds and removes content from the DOM when its condition is triggered.

Syntax


@Component({
  selector: 'my-app',
  template: `
       <input id="textField" value="" />
       <span *ngIf="isInvalid">First name is Invalid</span>
    `
})
export class AppComponent {
    if (txtField == ''){
        isInvalid = true
    }
}

Component Styles

Styling components in Angular is different than traditional styling.

Inline Syntax


@Component({
  selector: 'app-comp',
  template: `
    
Hello Angular

  `,
  styles: [`
    h1 { font-weight: normal; }
    h2 { font-weight: bold; }
   `]
})
export class HeroAppComponent {
/* . . . */
}

External Syntax


@Component({
  selector: 'app-comp',
  template: `
    
Hello Angular

  `,
    styleUrls: ['app.component.css']
})
export class HeroAppComponent {
/* . . . */
}

CSS Encapsulation


@Component({
  selector: 'app-comp',
  template: `
    
Hello Angular

  `,
  styles: [`

Angular CLI

The Angular CLI makes it very easy to create a new application, create new components, and test locally.

Angular CLI Benefits

  • Generates Project Structure
  • Creates Most Files
  • Runs your development server
  • Follows best practices

Installing Angular CLI


    npm install -g @angular/cli

This installs all the dependencies needed to run the Angular CLI from the command line.

Creating a New Project


    ng new MY-PROJECT-NAME 
    cd MY-PROJECT-NAME
    ng serve

Flags

The new command has several useful flags available in it.

  • --dry-run
  • --prefix
  • --routing
  • --style

Interfaces In the CLI

Interfaces provide the easiest way to provide consistency across your application.

Interface Generation


ng generate interface my-interface-name

#or

ng g i my-interface-name

Only generates .ts file.

Interface Structure


export interface MyInterfaceName {
    id:string;
    name: string;
    description:string;
}

Interface Usage


import { Component } from '@angular/core';

import { MyInterfaceName } from './my-interface-name';

@Component({
  selector: 'my-app',
  template: `
`
})
export class AppComponent { 
    MyItem: MyInterfaceName;
}
