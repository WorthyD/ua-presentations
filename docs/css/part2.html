Backgrounds

Changes the background appearance of the element.

  • background-color
  • background-image
  • background-repeat repeats the background image.
    • repeat-x repeats the image across the x axis
    • repeat-y repeats the image across the y axis
    • no-repeat makes the image only appear once
  • background-position Accepts two values. First the x and then the y. You specify these by using these values.
    • left|right|center  top|center|bottom
    • 1px 1px; - This would be 1px off the top and 1px off the left
    • x% y%

Borders

This CSS property defines the borders around an element.

  • border-width accepts pixel value
  • border-style
    • solid the most common
    • dashed
    • dotted
    • Many others
  • color accepts any valid color
border width also accepts thin medium and thick, but px is the most desired

Examples


    .fig1{
        border-color: red;
        border-style: solid;
        border-width:5px;
    }
    .fig2{
        border-color: navy;
        border-style: dotted;
        border-width:10px;
    }
Red border that is 5px thick.
Fig2. Dotted border

That's a lot of code. Can we shorten it?


    .fig3 {
        border: 5px solid red;
    }
    .fig4 {
        border:10px dotted navy;
    }
Red border that is 5px thick.
Fig4. Dotted border

Modify a specific border


    .fig6 {
        border-left: 5px solid red;
    }
    .fig7 {
        border: 10px dotted navy;
        border-right-style:solid;
        border-right-width:5px;
    }

Fig 7 - My right is different

More specificity


    .fig8 {
        border: 5px solid red;
        border-bottom:none;
    }
    .fig9 {
        border-bottom:none;

Margins

CSS Margins add spacing around an element.

  • margin-left
  • margin-top
  • margin-right
  • margin-bottom

Margin Shorthand


.fig10 {
    /* Applies 5px to all 4 sides */
    margin: 5px;
}
.fig11 {
    /* Applies 5px to all 4 sides */
    margin: 5px 5px 5px 5px;
}
.fig12{
    /*Applies 10px to top and bottom
      and 30px left and right*/
    margin:10px 30px;
}

Display

The CSS display property is used to change the default rendering style of an element.

  • inline - span, em, strong
  • block - div, p, h1-h6
  • none - hides the element
  • inline-block - Renders block, but stays in a single line.

You can change any element to display any type.

Set inline to block, but not block to inline.
